## Best Practices

### Timing

-   Post during peak hours (9-11 AM or 7-9 PM EST)
-   Post on weekdays for better engagement
-   Consider posting multiple times with different angles

### Formatting

-   Use emojis sparingly (2-3 max)
-   Include line breaks for readability
-   Use thread format for longer content
-   Pin the tweet with video

### Engagement

-   Tag relevant accounts: `@IndieFun`, `@solana`, `@SolanaGaming`
-   Reply to your own tweet with links
-   Use relevant hashtags (3-5 max)
-   Include a clear call-to-action

### Visuals

-   Use video thumbnail as preview image
-   Include project logo/branding
-   Add text overlay on video (if needed)
-   Ensure mobile-friendly format
